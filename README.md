🛒 Market Stock Management System
Bu proje, bir marketin stok, satış ve personel süreçlerini yönetmek için geliştirilmiş web tabanlı bir Yönetim Paneli (Admin Dashboard) uygulamasıdır.

🚀 Kullanılan Teknolojiler
Backend: ASP.NET Core MVC (C#)

Database: Microsoft SQL Server

ORM: Entity Framework Core

Frontend: HTML5, CSS3, JavaScript (Dashboard UI)

Mimari: MVC (Model-View-Controller)

✨ Temel Özellikler
Yönetim Paneli: Azalan stokların, son yapılan satışların ve alışların anlık takibi.

Ürün Yönetimi: Barkod tabanlı ürün ekleme, listeleme, güncelleme ve stok durumu kontrolü.

Satış Geçmişi: Yapılan tüm satışların miktar, birim fiyat, toplam tutar, tarih ve kasiyer bazlı detaylı raporlanması.

Kasiyer Takibi: Sisteme tanımlı kasiyerlerin ve bağlı oldukları kasaların yönetimi.

Güvenli Giriş: Admin yetkilendirmesi ile market yönetim paneline güvenli erişim.
